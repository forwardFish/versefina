# Sprint 2: 世界、撮合、账本、日循环

## Sprint 目标

建立统一市场世界底座，实现 Agent 的自动交易循环，跑通"世界运行 → Agent 决策 → 撮合 → 账本更新 → 绩效记录"完整闭环。

## Epic 划分

### Epic 2.1: World Foundation
- S2-001: 统一市场世界初始化
- S2-002: 行情数据接入与日线价格服务

### Epic 2.2: Matching & Execution
- S2-003: 撮合引擎与订单执行
- S2-004: 账本与持仓管理

### Epic 2.3: Daily Loop
- S2-005: Agent 决策接口与信号生成
- S2-006: 日循环调度器
- S2-007: 绩效计算与快照

## 关键里程碑

1. **世界底座**（S2-001 ~ S2-002）：统一世界和行情数据就绪
2. **交易闭环**（S2-003 ~ S2-004）：撮合和账本可用
3. **自动运行**（S2-005 ~ S2-007）：日循环自动驱动所有 Agent

## 验收标准

- 世界能够按交易日历自动推进
- Agent 能够基于画像生成交易信号
- 撮合引擎能够正确执行订单并更新账本
- 每日绩效快照正确记录

## 技术栈

- 后端：FastAPI + Python 3.11
- 行情数据：akshare（免费）
- 调度：APScheduler 或 Celery Beat
- 数据库：PostgreSQL

## 依赖

- Sprint 1 全部完成（Agent 创建 API）
