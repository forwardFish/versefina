# Sprint 8 规划

## Sprint 名称

事件产品化与 MVP 验收

## Sprint 目标

把前面三个 Sprint 的事件推演能力包装成 Versefina 可用的只读产品闭环：完成事件卡 projection 和查询、事件输入与推演卡页、复盘页、批量验证和 MVP 验收包。

## 做什么

- [ ] 完成 Event Report Card projection 与查询 API
- [ ] 完成事件输入页和事件推演卡页
- [ ] 完成复盘页与 evidence ledger 展示
- [ ] 完成批量验证基线与 MVP 验收包

- [ ] 不做复杂多用户协作权限体系
- [ ] 不做开放 Agent 市场
